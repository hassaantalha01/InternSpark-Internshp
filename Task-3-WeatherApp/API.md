## 🌐 API Integration

This Weather App uses a weather API to fetch real-time weather information and display it dynamically to users.

The application communicates with the API using JavaScript's **Fetch API** to send requests and receive weather data in JSON format.

### 🔄 How It Works

1. User enters a city/location name.
2. The application sends a request to the weather API.
3. The API processes the request and returns real-time weather data.
4. JavaScript extracts the required information from the JSON response.
5. The UI updates dynamically to display:

- 🌡️ Temperature
- ☁️ Weather condition
- 💧 Humidity
- 🌬️ Wind speed
- 📍 Location details

### 🛠️ Concepts Implemented

- API requests using Fetch API
- Asynchronous JavaScript (`async/await`)
- JSON data handling
- Error handling for invalid locations
- Dynamic DOM manipulation

### 📌 Purpose

The API integration makes the application interactive and allows users to access updated weather information without refreshing the page.

This project helped me gain practical experience in working with external APIs and building real-world frontend applications.
